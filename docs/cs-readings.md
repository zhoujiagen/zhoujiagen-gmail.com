# Computer Science Reading List

## Computer Systems

- Randal E. Bryant, David O'Hallaron. **深入理解计算机系统**. 机械工业出版社, 2017.
- W. Richard Stevens, Stephen A. Rago. **UNIX环境高级编程**. 人民邮电出版社, 2014.
- W. Richard Stevens. **TCP/IP详解, 卷1: 协议**. 人民邮电出版社, 2016.

## Programming Languages

- Martin Fowler. **领域特定语言/Domain Specific Languages**. 机械工业出版社, 2013.
- Terence Parr. **编程语言实现模式/Language Implementation Patterns**. 华中科技大学出版社, 2012.

## Data Structures and Algorithms

- Robert Sedgewick. **Algorithms in C**. Addison-Wesley, 2006.
- Robert Sedgewick, Kevin Wayne. **Algorithms**. Pearson Education, 2011.
- Maurice Herlihy, Nir Shavit. **The Art of Multiprocessor Programming**. Elsevier, 2012.

## Data Applications

- Ben Forta. **MySQL必知必会**. 人民邮电出版社, 2009.
- Martin Kleppmann. **Designing Data-Intensive Applications**. O'Reilly Media, 2017.
- Tyler Akidau, Slava Chernyak, Reuven Lax. **Streaming Systems**. O'Reilly Media, 2017.
- Christian Cachin, Rachid Guerraoui, Rodrigues Luís. **Introduction to Reliable and Secure Distributed Programming**. Springer Science & Business Media, 2011.
